Șiruri de Caractere (Strings)

Lecția 5 | 10-14 ani | Nivel: Incepator - Intermediar

1. Ce este un Șir de Caractere?

În Python, un șir de caractere (string) reprezintă o secvență ordonată de caractere — litere, cifre, spații sau simboluri speciale. Se definesc folosind ghilimele simple (' ') sau duble (" ").

print('Salut') print("Salut") # Triple ghilimele pentru mai multe linii text_lung = """Acesta este un string pe mai multe rânduri."""

2. Accesarea Caracterelor și Slicing

Fiecare caracter are un index numeric, începând de la 0. Indexul -1 reprezintă ultimul caracter.

text = "Python" print(text[0]) # 'P' print(text[-1]) # 'n' # Slicing: [start:stop:step] print(text[0:3]) # 'Pyt' print(text[::2]) # 'Pto'

3. Modificarea Stringurilor

Stringurile sunt imutabile. Operațiile returnează întotdeauna un nou string.

txt = " Salut Lume " print(txt.upper()) # " SALUT LUME " print(txt.strip()) # "Salut Lume" print(txt.replace("Lume", "Python")) # " Salut Python "

4. Concatenarea Stringurilor

Unirea stringurilor se face cu operatorul +, iar repetarea cu *.

print('Buna' + ' ' + 'ziua') # 'Buna ziua' print('Ha' * 3) # 'HaHaHa'

5. Formatarea Stringurilor

Cea mai recomandată metodă este f-string (Python 3.6+).

nume = "Alex" varsta = 12 print(f'Numele meu este {nume} si am {varsta} ani')

6. Caractere Speciale (Escape)

Secvențe ce încep cu backslash (\) pentru caractere ce nu pot fi scrise direct.

print("Linie1\nLinie2") # Rând nou print("Tab\torizontal") # Spațiu Tab print("Ghilimele: \"Python\"")

7. Metode Utile

Metode frecvente: len(), find(), startswith(), count(), isdigit().
txt = "Python123" print(len(txt)) # 9 print(txt.isdigit()) # False (are și litere)
