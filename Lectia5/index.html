1. Ce este un Șir de Caractere?

În Python, un string reprezintă o secvență ordonată de caractere — litere, cifre, spații sau simboluri speciale.

Se definesc folosind ghilimele simple (' ') sau duble (" "). Python suportă și ghilimele triple pentru texte pe mai multe linii.

text1 = 'Salut' text2 = "Salut" multi_line = """Acesta este un text care se întinde pe mai multe rânduri."""

2. Accesarea Caracterelor și Slicing

Fiecare caracter are un index numeric, începând de la 0. Indexul -1 reprezintă ultimul caracter.

limbaj = "Python" print(limbaj[0]) # 'P' print(limbaj[-1]) # 'n'

Slicing (Felierea)

Slicing-ul extrage un subșir folosind sintaxa string[start:stop:step]. Valoarea stop nu este inclusă.

text = "Python" print(text[0:3]) # 'Pyt' print(text[::2]) # 'Pto' (sari peste o literă)

3. Modificarea Stringurilor

Stringurile sunt imutabile — nu pot fi modificate direct. Metodele returnează un string nou.

MetodăEfectExemplu
upper()Litere MARI"salut" -> "SALUT"
lower()Litere mici"SALUT" -> "salut"
strip()Șterge spații" hey " -> "hey"
replace()Înlocuiește"mere" -> "pere"

4. Concatenarea Stringurilor

Unirea stringurilor se face cu operatorul +, iar repetarea cu *.

salut = "Buna" + " " + "ziua" # "Buna ziua" ras = "Ha" * 3 # "HaHaHa"

5. Formatarea Stringurilor

Cea mai modernă metodă este f-string. Prefixăm textul cu f și folosim acolade {}.

nume = "Alex" varsta = 12 print(f"Numele meu este {nume} și am {varsta} ani.")

6. Caractere Speciale (Escape)

Secvențele care încep cu backslash (\) reprezintă caractere speciale.

print("Linie1\nLinie2") # Rând nou print("Tab\torizontal") # Spațiu mare print("Ghilimele: \"Python\"")

7. Metode Utile

Python include un set bogat de funcții pentru a analiza textul:

len() ne spune lungimea totală a textului (inclusiv spațiile).

parola = "secret123" print(len(parola)) # 9 print(parola.isdigit()) # False print(parola.find("1")) # 6 (poziția cifrei 1)
