25 Exerciții Python - Lecția 1

1. Afișează numele tău dintr-o variabilă:
Vezi rezolvarea
# Stocam textul intr-o variabila si il afisam
nume = "Alex"
print(nume)
2. Afișează numele și prenumele (Popescu Ion):
Vezi rezolvarea
# Lipim doua variabile text cu spatiu intre ele
nume = "Popescu"
prenume = "Ion"
print(nume + " " + prenume)
3. Salut, lume (virgulă lipită):
Vezi rezolvarea
# Concatenam virgula de primul cuvant pentru a evita spatiul automat
sir1 = "Salut"
sir2 = "lume"
print(sir1 + ",", sir2)
4. Transformă "python" în litere mari:
Vezi rezolvarea
# Metoda upper transforma textul in majuscule
txt = "python"
print(txt.upper())
5. Lungimea textului "Scoala":
Vezi rezolvarea
# Functia len numara caracterele din sir
print(len("Scoala"))
6. Suma a două numere (10 și 3):
Vezi rezolvarea
# Operatie simpla de adunare intre intregi
print(10 + 3)
7. Înmulțirea 4.5 cu 2:
Vezi rezolvarea
# Inmultirea unui numar float cu un intreg
print(4.5 * 2)
8. Partea întreagă a împărțirii (10 la 3):
Vezi rezolvarea
# Operatorul // returneaza doar catul
print(10 // 3)
9. Restul împărțirii (15 la 4):
Vezi rezolvarea
# Modulo (%) returneaza restul impartirii
print(15 % 4)
10. 2 la puterea 10:
Vezi rezolvarea
# Operatorul ** este folosit pentru ridicarea la putere
print(2 ** 10)
11. Transformă 5 în float:
Vezi rezolvarea
# Convertim numarul intreg in numar cu virgula
print(float(5))
12. Adună textul "100" cu numărul 50:
Vezi rezolvarea
# Convertim textul in intreg inainte de adunare
print(int("100") + 50)
13. Elimină zecimalele lui 25.7:
Vezi rezolvarea
# Functia int taie zecimalele fara rotunjire
print(int(25.7))
14. Citește un număr și afișează dublul lui:
Vezi rezolvarea
# Citim input, convertim in int si inmultim cu 2
n = int(input("Introdu numar: "))
print("Rezultatul este:", n * 2)
15. Afișează tipul variabilei x = 10.5:
Vezi rezolvarea
# type() ne arata clasa din care face parte variabila
x = 10.5
print(type(x))
16. Creează o variabilă booleană:
Vezi rezolvarea
# Atribuim valoarea True unei variabile
adevar = True
print(adevar)
17. Verifică dacă 10 este mai mare ca 5:
Vezi rezolvarea
# Afisarea rezultatului unei comparatii logice
print(10 > 5)
18. Verifică dacă "A" este egal cu "a":
Vezi rezolvarea
# Python este case-sensitive (face diferenta intre litere)
print("A" == "a")
19. Verifică dacă 7 este diferit de 8:
Vezi rezolvarea
# Operatorul != verifica inegalitatea
print(7 != 8)
20. Inversează valoarea True:
Vezi rezolvarea
# Operatorul not neaga valoarea logica
print(not True)
21. Schimb de variabile (Swap):
Vezi rezolvarea
# Interschimbam x si y intr-o singura linie
x, y = 5, 10
x, y = y, x
print(x, y)
22. Perimetru dreptunghi (L=20, l=10):
Vezi rezolvarea
# Calculam suma laturilor inmultita cu 2
L, l = 20, 10
print(2 * (L + l))
23. Media a trei note:
Vezi rezolvarea
# Adunam notele si impartim la numarul lor
print((8 + 9 + 10) / 3)
24. Afișează "Preț: 100 lei":
Vezi rezolvarea
# Combinam textul cu variabila numerica
p = 100
print("Pret:", p, "lei")
25. Calcul profit (Venit 5000, Cost 3200):
Vezi rezolvarea
# Scadem cheltuielile din venitul brut
print(5000 - 3200)
