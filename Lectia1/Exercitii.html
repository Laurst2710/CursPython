25 Exerciții Python - Lecția 1

1. Afișează numele tău dintr-o variabilă: 
# Versiunea 1.1: Definim variabila si o afisam
nume = "Alex"
print(nume)
2. Afișează numele și prenumele: 
# Versiunea 1.1: Lipim numele de prenume cu un spatiu
nume = "Popescu"
prenume = "Ion"
print(nume + " " + prenume)
3. Salut, lume (virgulă lipită): 
# Versiunea 1.1: Lipim virgula de primul sir pentru a evita spatiul automat
sir1 = "Salut"
sir2 = "lume"
print(sir1 + ",", sir2)
4. Litere mari: 
# Versiunea 1.1: Utilizam functia upper pentru majuscule
txt = "python"
print(txt.upper())
5. Lungime text: 
# Versiunea 1.1: Numaram caracterele cu len
print(len("Scoala"))
6. Sumă (10 și 3): 
# Versiunea 1.1: Adunarea a doua numere intregi
print(10 + 3)
7. Produs float: 
# Versiunea 1.1: Inmultirea cu un numar zecimal
print(4.5 * 2)
8. Câtul împărțirii: 
# Versiunea 1.1: Aflarea catului fara rest
print(10 // 3)
9. Restul împărțirii: 
# Versiunea 1.1: Operatia modulo pentru rest
print(15 % 4)
10. Putere: 
# Versiunea 1.1: Ridicarea la putere
print(2 ** 10)
11. Conversie float: 
# Versiunea 1.1: Transformarea unui intreg in zecimal
print(float(5))
12. Text în număr: 
# Versiunea 1.1: Convertim textul pentru a putea face calcule
print(int("100") + 50)
13. Eliminare zecimale: 
# Versiunea 1.1: Fortam transformarea in intreg
print(int(25.7))
14. Input și dublu: 
# Versiunea 1.1: Citim de la tastatura si dublam valoarea
n = int(input("Numar: "))
print("Rezultatul este:", n * 2)
15. Tip variabilă: 
# Versiunea 1.1: Identificam tipul de date
x = 10.5
print(type(x))
16. Boolean: 
# Versiunea 1.1: Alocarea unei valori de adevar
v = True
print(v)
17. Comparație: 
# Versiunea 1.1: Verificarea unei inegalitati
print(10 > 5)
18. Egalitate: 
# Versiunea 1.1: Compararea a doua siruri
print("A" == "a")
19. Diferit: 
# Versiunea 1.1: Verificarea operatorului diferit
print(7 != 8)
20. Negare: 
# Versiunea 1.1: Inversarea valorii booleane
print(not True)
21. Schimb variabile: 
# Versiunea 1.1: Interschimbarea valorilor (swap)
x, y = 5, 10
x, y = y, x
print(x, y)
22. Perimetru: 
# Versiunea 1.1: Calculul perimetrului unui dreptunghi
L, l = 20, 10
print(2 * (L + l))
23. Medie: 
# Versiunea 1.1: Media aritmetica a trei numere
print((8 + 9 + 10) / 3)
24. Lipire text și număr: 
# Versiunea 1.1: Afisarea unui mesaj combinat
p = 100
print("Pret:", p, "lei")
25. Economii: 
# Versiunea 1.1: Scaderea cheltuielilor din venit
print(5000 - 3200)
