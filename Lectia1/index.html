1. Schimbarea valorii unei variabile

scor = 0 print(scor) # Afișează: 0 scor = 500 print(scor) # Afișează: 500 (aceeași cutie, valoare nouă) scor = scor + 100 print(scor) # Afișează: 600

2. Casting: float()

Transformă în număr decimal (cu virgulă):

nota = float("9.5") print(nota + 0.5) # Rezultat: 10.0 scor = 100 print(float(scor)) # Rezultat: 100.0 (adaugă .0)
