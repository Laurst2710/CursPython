Variabile, Tipuri de date și Casting

Introducere Variabile Tipuri Casting

Ce este Python?

Python este un limbaj de programare prietenos, creat să fie ușor de citit. Este folosit de companii precum Google, NASA și în dezvoltarea Inteligenței Artificiale.

De ce Python? Este mult mai scurt decât alte limbaje. Pentru a afișa un mesaj, scrii doar o linie de cod.

1. Variabile

O variabilă este ca o cutie etichetată în care stocăm informații.

# Crearea unei variabile nume = "Andrei" varsta = 14 print(nume) # Putem schimba valoarea mai târziu varsta = 15

Reguli pentru nume:

  • Trebuie să înceapă cu o literă sau underscore (_).
  • Nu pot începe cu o cifră.
  • Pot conține doar caractere alfanumerice și underscore (A-z, 0-9, și _).

3. Casting (Conversia tipurilor)

Casting-ul este procesul prin care transformăm o valoare dintr-un tip în altul.

int()
float()
str()

Funcția int()

Transformă în număr întreg. Dacă transformi un float, acesta va pierde zecimalele (nu rotunjește).

print(int(9.8)) # Rezultat: 9

Funcția float()

Transformă în număr decimal. Adaugă .0 la finalul numerelor întregi.

nota = float("9.5") print(nota + 0.5) # Rezultat: 10.0 scor = 100 print(float(scor)) # Rezultat: 100.0

Funcția str()

Transformă orice valoare în text. Util pentru concatenare.

varsta = 14 print("Am " + str(varsta) + " ani")
